## Project Overview

This repository contains TypeScript exercises for the **ALX Frontend JavaScript program**.  
The tasks demonstrate:

- Creating and using interfaces  
- Extending interfaces  
- Working with classes  
- Advanced types and type guards  
- String literal types  

All tasks are compiled using **Webpack** and written in **TypeScript** with strict type safety.

- Node.js (v20 recommended)
